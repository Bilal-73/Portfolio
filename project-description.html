Project Details

Amazon Review Classification

Sentiment analysis using NLP to classify Amazon reviews into positive or negative. This project applies text preprocessing, vectorization, and machine learning techniques to perform sentiment classification on product reviews. The model is trained to distinguish between positive and negative sentiments, providing useful insights into customer feedback.

Image Segmentation with U-Net

This project uses a U-Net architecture for segmenting medical images. U-Net is a deep learning model particularly effective for pixel-wise classification in medical imaging, such as segmenting tumors, organs, or other key structures from medical scans. This project highlights the importance of data augmentation, loss functions, and model evaluation in deep learning applications.
