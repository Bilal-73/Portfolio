YOLOv8 Real-Time Video Detection with Flask

One-liner: A web app that performs real-time object detection on uploaded videos using YOLOv8, and streams results in-browser.

üß† Project Overview

This project demonstrates real-time object detection on video files using the YOLOv8 model by Ultralytics. Built with Flask and OpenCV, it enables users to upload a video, process it frame-by-frame for object detection, and stream the annotated video directly on the web interface. It serves as a foundation for integrating AI-powered vision systems into web platforms for educational demos, surveillance, or smart monitoring applications.

üöÄ How It Works

  1. User uploads a video file through the web interface.
  2. The server stores the video and begins reading it frame-by-frame using OpenCV.
  3. Each frame is passed through the YOLOv8 model for object detection.
  4. Bounding boxes and labels are drawn on detected objects.
  5. Annotated frames are streamed back to the browser using MJPEG format.
